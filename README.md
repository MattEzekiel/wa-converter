## Installation

You can install the package via npm:
### npm
```bash
npm install whatsapp-conveter
```

### yarn
```bash
yarn install whatsapp-conveter
```

### pnpm
```bash
pnpm add whatsapp-conveter
```

## Usage
### Import 
````ts
import WA from 'wa-converter';
````

### Add to your code
```ts
formatWhatsAppUrl('+123456489', 'Hello friend');
```
